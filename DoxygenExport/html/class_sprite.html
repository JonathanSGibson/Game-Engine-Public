Constructor for the sprite component.

Sets various required variables of the component based on the parameters

Parameters
owner_The object that owns this component
fileDirectoryThe file directory of the image to be loaded as the sprite
renderLayer_The layer the sprite will be rendered on, defaulted to 0
useTransparencyWhether or not the sprite will remove transparent (keyed as 255, 0, 255) backgrounds, defaulted to false
Note
Also outputs errors to the console if the texture or surface cannot be loaded correctly
